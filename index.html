How to use

  • Start typing a patient’s needs (plain language) in the search box — results update automatically.
  • Use the category tiles to browse visually if you’re not sure where to start.
  • Open Full description on any card to view the complete text from the source document.
  • Select services with Add, then edit wording/contact details in the right panel if needed.
  • Full service descriptions are always included in the patient output.
  • Use Copy, Print, or Download PDF to share. Printing will show full URLs.
  • Hyperlink / Plain text controls whether links are copied as clickable links or as written-out URLs.
